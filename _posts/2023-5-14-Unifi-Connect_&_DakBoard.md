# Tl;dr

Unifi Connect is a new product line that aims at providing a digital signage solution without subscriptions or dependencies on the cloud should your needs dictate staying off cloud solutions. Currently, the Connect application can be run on Unifi Cloud Key Gen 2. See the [Unifi Connect](https://www.ui.com/unifi/unifi-connect/) page for more information.

# My Use case

As I menioned in my original post on [Unifi Connect & HomeAssistant](https://blog.chriszavala.com/Unifi-Connect_&_HomeAssistant/) I wanted to use the Unifi Connect application to display information on a digital signage solution. I had previously used [DakBoard](https://dakboard.com/) for this purpose and was happy with the results. I had also used [HomeAssistant](https://www.home-assistant.io/) to display information on the DakBoard. I wanted to see if I could use the Unifi Connect application to display information from HomeAssistant.
